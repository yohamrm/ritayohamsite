Canvas Redesign Feature Project

Project Overview

The objective for this project was to identify a feature within a product that needs to be improved on and find a design solution. I decided to make improvements to the announcements and messages features of Canvas, a platform where all materials for classes in higher level education institutions can be posted. Throughout this project, I went through the process of identifying the problem and pain points through user research and coming up with a redesign recommendation to solve the problems these features pose.

The Process

I first defined the users and goals for this project. I then analyzed the application and identified the problem within Canvas. Next, I conducted user research through surveys to confirm that this is a problem among Canvas users and to get suggestions for how the Announcements and Messages features can be improved. Finally, I identified pain points and wrote my design problem statement. After gathering all this information, I started working on the redesign of the features. I did this through the creation of user flows, low fidelity mockups, and high fidelity mockups. I gathered all my data and thoughts into a conclusion and provided my recommendation for a redesign of the features. You can view the entire project below.
