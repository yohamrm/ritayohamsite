Roots Run Deep Internship

Roots Run Deep Logo

Project Overview

I was a Design Intern for the Shenandoah Valley Black Heritage Project from August 2020 through May 2021. My role was to help create a website, logo, and print materials for a project they were working on called Roots Run Deep. The goal of the project was to create a walking tour of African American history in the City of Harrisonburg. I worked alongside a History Graduate student at JMU in order to complete this project. We built the website through Weebly in order for the Director of the SVBHP to easily update it since that is her preferred site builder. It has been updated by her since I last worked on the website.

The Process

We started the process of building the website by creating a sitemap, identifying all the different pages and functions that the website needed to have. After that, I expressed my ideas for the logo design and created that for the project. We then built the website and continued to update it as we learned more information about historical sites that should be added to it. There have been updates made to the website since my internship ended, but I helped build the foundation of the website. I also helped with build a preliminary layout of the brochure that was used for the tour.
